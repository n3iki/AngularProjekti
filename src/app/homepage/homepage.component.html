Moe!

Tämä sivusto on osa LAB-ammattikorkeakoulun "Front-end työkalut" -kurssia. Valikosta löytyy kaikki kurssin sisältämien harjoitustöiden ratkaisut.

Sivuston luomiseen on käytetty seuraavia tekniikoita:

  • HTML5
  • CSS
  • Angular
  • TypeScript
  • BootStrap
  • Firebase
  • Git(Hub)

Sivusto on skaalautuva, mutta ei mobiilioptimoitu. Ulkoasukin on vielä rujo, hiotaan sitä mukaa kun ehtii. Sivuston kansiorakenne on myös todella sekava, pitää ottaa selvää onko kansiointi jälkikäteen järkevää. Koodin kommentointi myös jokseekin puutteellista, en tosin tiedä onko harjoitustyössä syytä nähdä vaivaa perusteelliseen kommentointiin.

Hauskana kuriositeettina; bannerin norppa on Stable Diffusion UI -tekoälyn luoma!
